# Breviary Latin→中文 LOCKED Glossary

Authoritative term-lock for translating the Roman Breviary (Divine Office, 1960)
from **Latin** into Chinese. Every Latin term/name/formula below is rendered
**exactly** as given, so the same word is identical across all chunks. This file
overrides individual translator judgment — but never the absolute rules
(God = 神, Allelúia = 阿肋路亚, Bible names per CUV). Read together with
`robo-hieronymus prompt.md` and `pilot/ADDENDUM.md`.

Paradigm reminder: ecumenical-most-common term. Protestant/CUV default; Catholic
terms only where the concept is distinctively Catholic (圣事, papal names,
saints with no Protestant currency). **Bible figures = CUV/Protestant names**
even inside Catholic phrases. **Church Fathers/saints = ecumenically-most-common
name** (usually Protestant). Deuterocanonical books/figures = 思高 names.

## 10. Resolved decisions (user-confirmed — LOCK)

These were borderline; the user has decided. Apply verbatim everywhere.

1. **John Chrysostom (Ioánnes Chrysóstomus)** → **金口若望**.
2. **Gregory the Great (Gregórius Papæ)** → **额我略** (papal name = Catholic, per
   robo rule). St. Gregory the Great = 圣大额我略.
3. **Leo the Great (Leo Papæ)** → **良** (papal name = Catholic). St. Leo the Great
   = 圣大良.
4. **Mother of God (Dei Génetrix / Deípara)** → **神之母** (consistent with God=神;
   天主之母 is forbidden). Where the Latin is just `Mater`/`Genitrix` meaning "the
   Blessed Mother" (not the formal title), 圣母 is acceptable.
5. **Confessor** (saint-category, non-martyr; and the homily/title sense) →
   **宣信者**. So `Confessor Pontifex` = 宣信者主教; `Commune Confessoris non
   pontificis` = 非主教宣信者共通经文; etc.
6. **Church Father transliterations** (no CUV-fixed form) — LOCK as:
   Ambrose=安波罗修, Bede=比德, Bernard=伯尔纳, Basil=巴西尔, Hilary=希拉里.
   (Augustine=奥古斯丁, Jerome=耶柔米, Athanasius=亚他拿修 already locked in §6.)
7. **Eve** → name = **夏娃**. In hymns with the Áve/Éva (Ave/Eva) wordplay the pun
   is untranslatable — render the sense and note it in `_notes`.
8. **Nebuchadnezzar (Nabuchodónosor)** → **尼布甲尼撒** (CUV) everywhere, EXCEPT
   inside verbatim 友弟德传 (Judith) 思高 quotations, where the fetched 思高 text reads
   拿步高.
9. **"feria" weekday naming** → **周一/周二/…/周六** (Feria II=周一 … Feria VII/
